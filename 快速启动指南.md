# 🚀 Nano Banana 快速启动指南

## 🎯 立即体验

### 1. 启动应用
```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

### 2. 访问应用
打开浏览器访问：`http://localhost:3001`

### 3. 体验流程
```
🎁 看到右上角显示：免费试用: 3/3 次 (无需登录)
🎨 直接输入描述开始生成，无需注册
📸 生成完成后可下载图片
🔄 第4次使用时会优雅地提示登录
```

## 🎨 界面特点

### 匿名用户界面
- 右上角：`🎁 免费试用: 3/3 次 (无需登录)`
- 生成按钮：`🎨 免费生成 (1 张) 💎 3/3 免费`
- 功能限制：只能选择1张图片

### 注册用户界面
- 右上角：用户名 + 积分数量 + 专业版标签
- 生成按钮：`🎨 开始生成 (多张) 💎 N 积分`
- 功能升级：可选择2-4张图片

### 付费用户界面
- 右上角：用户名 + `💎 ∞` + 无限版标签
- 生成按钮：`🎨 开始生成 (批量) 💎 ∞ 积分`
- 高级功能：⭐ 高级功能已启用提示
