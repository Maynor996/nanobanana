# 📧 邮件验证配置指南

## 🎯 概述

为了实现真实的邮箱验证登录功能，您需要配置邮件服务。本指南提供了多种配置方案。

## 🚀 快速配置

### 方案一：QQ邮箱 SMTP（推荐）

1. **开启QQ邮箱SMTP服务**
   - 登录QQ邮箱网页版
   - 设置 → 账户 → POP3/IMAP/SMTP/Exchange/CardDAV/CalDAV服务
   - 开启"POP3/SMTP服务"或"IMAP/SMTP服务"
   - 获取授权码（不是QQ密码）

2. **配置环境变量**
   ```bash
   # 创建 .env.local 文件
   SMTP_HOST=smtp.qq.com
   SMTP_PORT=465
   SMTP_USER=你的QQ邮箱@qq.com
   SMTP_PASS=你的QQ邮箱授权码
   ```

### 方案二：Gmail SMTP

1. **开启Gmail应用专用密码**
   - 开启两步验证
   - 生成应用专用密码

2. **配置环境变量**
   ```bash
   GMAIL_USER=your_email@gmail.com
   GMAIL_APP_PASSWORD=your_gmail_app_password
   ```

### 方案三：其他邮箱服务

```bash
# 163邮箱
SMTP_HOST=smtp.163.com
SMTP_PORT=465
SMTP_USER=your_email@163.com
SMTP_PASS=your_authorization_code

# 126邮箱
SMTP_HOST=smtp.126.com
SMTP_PORT=465
SMTP_USER=your_email@126.com
SMTP_PASS=your_authorization_code

# 企业邮箱
SMTP_HOST=smtp.exmail.qq.com
SMTP_PORT=465
SMTP_USER=your_email@yourcompany.com
SMTP_PASS=your_password
```

## 📋 常见问题

### Q: 邮件发送失败怎么办？

**A: 检查以下几点：**
1. 确认SMTP服务已开启
2. 确认授权码正确（不是登录密码）
3. 检查网络连接
4. 查看控制台错误信息

### Q: QQ邮箱授权码在哪里获取？

**A: 获取步骤：**
1. 登录QQ邮箱网页版
2. 设置 → 账户
3. 找到"POP3/IMAP/SMTP/Exchange/CardDAV/CalDAV服务"
4. 开启服务时会提供授权码

### Q: 可以使用其他邮件服务吗？

**A: 可以！** 支持所有SMTP服务：
- 阿里云邮件服务
- 腾讯云邮件服务  
- SendGrid
- AWS SES
- 自建邮件服务器

只需要配置相应的SMTP参数即可。
