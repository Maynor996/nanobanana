# 🍌 Nano Banana 界面说明

## 概述
基于提供的设计图，我们创建了一个现代化的暗黑主题 AI 图像生成界面，完全复刻了 Nano Banana 的设计风格。

## 🎨 设计特点

### 1. 暗黑主题
- 背景色：`#0a0a0a`（纯黑）
- 卡片背景：`#111111`（深灰）
- 边框色：`#333333`（暗灰）
- 主题色：`#10b981`（翠绿）

### 2. 双模式支持
- **📷 通过对话编辑图像**：上传图片后进行 AI 编辑
- **✨ 文生图模式**：通过文字描述生成图片

### 3. 快速风格选择
- 🔍 增强细节
- 🎨 艺术风格
- ✨ 动漫风格
- 📷 写实照片

### 4. 灵感启发
提供 8 个快速提示词标签：
- 风景、人像、建筑、艺术
- 科幻、自然、动物、创意

## 🚀 访问方式

### 方法 1：Next.js 应用
```bash
# 启动开发服务器
npm run dev

# 访问 Nano 界面
http://localhost:3000/nano
```

### 方法 2：独立 HTML 测试
直接在浏览器中打开：
```
nano-test.html
```

## ✨ 功能特性

### 1. 上传模式
- 支持拖拽上传
- 图片预览功能
- 支持多种格式（PNG, JPG, JPEG, WebP, GIF）
- 文件大小限制：10MB

### 2. 文生图模式
- 实时字数统计（最多 5000 字符）
- 快速提示词选择
- 详细的描述输入框

### 3. 生成控制
- 图片数量选择（1-4 张）
- 样式快速切换
- 积分显示（每张 4 积分）

### 4. 响应式设计
- 适配各种屏幕尺寸
- 流畅的动画效果
- 优雅的悬停状态
